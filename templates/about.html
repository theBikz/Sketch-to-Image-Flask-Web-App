ABSTRACT

Today, Technology have remarkable charm in the area of Computer Graphics and Vision. Producing absolute images from the poor hand-drawn sketches is a very demanding and laborious task in this area. Hand-drawn sketch recognition is widely used in sketch based image and video retrieval, manipulations and reorganizations. Most existing methods use the exact edge maps or depend on known pictures. In Sketch to image synthesis, we translate the sketches to realistic images with the use of a generative model. We put forward an image to image translation network that involves in producing a synthesized image from the input sketch via an adversarial process. We are implementing a novel Conditional Generative Adversarial Network (cGANs) which is an extension of Generative Adversarial Networks (GANs) which produces the images with some sort of conditions or attributes. The GANs are deep learning neural architecture for learning a generative neural network for image synthesis. In this work, the implementation of cGANs for synthesizing the images from hand-drawn sketches. We had given sketches of shoes and hats as the input to the model and the model produces realistic images of them as output. We had obtained a remarkable output. The performance of the model was evaluated by calculating the generator loss and the discriminator loss. The output of the proposed sketch to image translation network was excellent and appreciable.
